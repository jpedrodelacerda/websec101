# 2 - Fundamentos

Antes de começarmos, é necessário compreender como funciona o universo onde atuaremos até o fim do guia. Afinal de contas, não queremos que você saia daqui um(a) script kiddie. Portanto, nada mais justo que começar do começo. Veremos como funciona os seguintes tópicos:

  1. URL
  2. HTTP
  3. HTML
  4. CSS

Depois de ter uma base sólida desses conhecimentos, poderemos partir para a ação!
